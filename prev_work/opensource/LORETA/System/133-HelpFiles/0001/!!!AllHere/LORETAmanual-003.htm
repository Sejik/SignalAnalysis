Summary

 

This is an attempt to document the LORETA software package for the analysis of EEG. LORETA is free-academic research software, and it is constantly changing. This document will typically be incomplete and not always up to date.

The user is expected to have working knowledge in the field of EEG analysis, and to have experience using computers running the windows operating system.

It is the user’s responsibility to use this software correctly: There are infinite possibilities for the user to make mistakes and produce completely invalid results.

If you use this software, then you must comply with the license agreement, which essentially says:

1. No clinical use.

2. No commercial use.

3. Use at your own risk.

Some necessary (but not sufficient conditions) for the use of this software

 

Find and run the windows explorer. The windows explorer is not the internet explorer. Figure 1 shows the windows explorer in windows 7 operating system.

Figure 1: “Windows explorer” in windows 7 operating system

 

Set the options for the windows explorer so that it displays as in Figure 1. Note that this is a detailed view of folders and files, which shows explicitly the extension of the files.

 

It is the user’s task to organize all working folders and files.

 

It is the user’s task to know the location of the working folders and files.

 

For instance, after installation of the software, the example EEG/ERP folders and files will be placed on the desktop. A user should be able to locate this folder and any file within immediately. This knowledge is a prerequisite for making efficient use of this software. It is outside the scope of this manual to instruct the user on these tasks.

About electrodes and files containing electrode coordinates

 

In this software, files with electrode coordinates have extension *.sxyz. They are simple text files. They can be opened in any text editor. Examples of text editor programs: NOTEPAD and WORDPAD. However, “microsoft office excel” is not a text editor, and will lock the file as long as excel is running, and make it completely unusable to this software or any other programs.

 

For example, an electrode coordinates file may contain the following text information (exclude the start and end lines with “-----”):

-----

19

-26.81 84.06 -10.56 Fp1

29.41 83.74 -10.04 Fp2

-66.99 41.69 -15.96 F7

-48.05 51.87 39.87 F3

0.90 57.01 66.36 Fz

50.38 51.84 41.33 F4

68.71 41.16 -15.31 F8

-83.36 -16.52 -12.65 T7/T3

-65.57 -13.25 64.98 C3

0.23 -11.28 99.81 Cz

66.50 -12.80 65.11 C4

84.44 -16.65 -11.79 T8/T4

-71.46 -75.17 -3.70 P7/T5

-55.07 -80.11 59.44 P3

-0.87 -82.23 82.43 Pz

53.51 -80.13 59.40 P4

71.10 -75.17 -3.69 P8/T6

-28.98 -114.52 9.67 O1

26.89 -114.68 9.45 O2

-----

 

Note the simple structure. The first line contains the number of electrodes (19), followed by 19 lines, each line containing 4 items separated by space or tab: x y z ElectrodeName. The coordinates XYZ in millimeters correspond to the MNI coordinate system. The electrode names can be arbitrary, but in this case correspond to the 10/10 placement system. Information about the MNI coordinates can be found at:

http://imaging.mrc-cbu.cam.ac.uk/imaging/MniTalairach

In case the link is not permanent, here are the references quoted from that homepage:

J. Talairach and P. Tournoux, "Co-planar Stereotaxic Atlas of the Human Brain: 3-Dimensional Proportional System - an Approach to Cerebral Imaging", Thieme Medical Publishers, New York, NY, 1988.

J. C. Mazziotta and A. W. Toga and A. Evans and P. Fox and J. Lancaster, "A Probablistic Atlas of the Human Brain: Theory and Rationale for Its Development", NeuroImage 2:89-101, 1995.

A. C. Evans and D. L. Collins and B. Milner, "An MRI-based stereotactic atlas from 250 young normal subjects", Journal Soc. Neurosci. Abstr. 18: 408, 1992.

A. C. Evans and D. L. Collins and S. R. Mills and E. D. Brown and R. L. Kelly and T. M. Peters, "3D statistical neuroanatomical models from 305 MRI volumes", Proc. IEEE-Nuclear Science Symposium and Medical Imaging Conference, 1813-1817, 1993.

 

Some information about electrode names from standard electrode placement systems can be found at:

http://en.wikipedia.org/wiki/10-20_system_(EEG)

http://www.bem.fi/book/13/13.htm#03

 

Importance of electrode order

 

The order of the electrodes must correspond to the order of the electric potentials in the EEG/ERP files. Incorrect order is disastrous, making localization information meaningless.
