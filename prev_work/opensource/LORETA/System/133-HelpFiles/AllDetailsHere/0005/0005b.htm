Computing the transformation matrix

sLORETA

The sLORETA transformation matrix (file with extension “*.psinv”) is used in the conversion (computation) of scalp electric potential differences to standardized current density in the brain. It is also used in the conversion (computation) of EEG cross-spectra to the intracranial spectral power.

sLORETA is based on an appropriately standardized version of the minimum norm current density. The minimum norm current density is demonstrably known to misplace deep sources to the outer cortex. However, when this current density is appropriately standardized, exact, zero-error localization is achieved.

The technical details are explained in:

R.D. Pascual-Marqui. Standardized low resolution brain electromagnetic tomography (sLORETA): technical details. Methods & Findings in Experimental & Clinical Pharmacology 2002, 24D:5-12.

The calculation of the transformation matrix requires knowledge of the head model, which in this case is the MNI152 template, as explained in the Tutorial. The lead field (the forward operator), which contains the head model information, was calculated for a fixed set of 316 electrodes from the 5% system, as explained in the Tutorial. In order to calculate the transformation matrix for any electrode coordinates, the lead field is interpolated to the User’s electrode coordinates, which do not have to correspond to the 5% positions.

This interpolation step is a bit time consuming.

eLORETA

The new exact low resolution brain electromagnetic tomography (eLORETA) is an authentic inverse solution with exact, zero-error localization. This means that the distributed current density not only “explains” the measured scalp electric potentials, but it also has the exact, zero-error localization property.

In practice, sLORETA and eLORETA produce very similar localization results.

eLORETA transformation matrix files have extension “-eLORETA.spinv”

All technical details on eLORETA can be found at:
http://arxiv.org/abs/0710.3341
